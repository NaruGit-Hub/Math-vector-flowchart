難易度： 基本 標準 応用 発展 ｜ ▶ をタップで展開
  • ③ 終点範囲・軌跡を求める
    • 末尾: 「Pの動く範囲を図示せよ」／「Pの存在領域を求めよ」／「領域を求めよ」
    • 3A: 基本パターン（暗記）
      • OP = sa + tb のとき
      • ・s, t 実数全体 → 全平面
      • ・s+t = 1 → 直線AB
      • ・s+t = 1, s≧0,t≧0 → 線分AB
      • ・s+t ≦ 1, s≧0,t≧0 → △OAB周+内部
      • 空間も同様: s+t+u に拡張
    • 3A2: 基本パターン（空間）
      • OP = la + mb + nc のとき
      • ・l+m+n = 1 → 平面ABC
      • ・l+m+n = 1, 各≧0 → △ABC周+内部
      • ・l+m+n ≦ 1, 各≧0 → 四面体OABC周+内部
    • 3B: 一般の終点範囲（3手順型）
      • s（または t）を固定
      • 動く方の変数で線を作る
      • 固定した変数を動かして面に拡大
      • 💡 0≦a≦1, 1/2≦b≦1 のような長方形領域で使う
    • 3C: (s+t) を作って3-Aに帰着
      • OP = (s+t) × {(sa+tb)/(s+t)} と変形
      • 内側の点 Q は線分AB上を動く
      • P は OQ を (s+t) 倍した点 → 図示
      • 💡 「パラメータについて整理する」の発想
    • 3D: 始点を変えて簡単化
      • a, b, c の係数和が a+b+c=1 等のとき
      • 始点を A に統一（→AP, AB, AC で書き直す）
      • 制約条件も書き換え
      • 3-A に帰着
    • 3E: 斜交座標による解法
      • a, b 方向を斜交軸とみなす
      • (s, t) 平面で不等式を図示
      • 元の図に戻す
      • 💡 答案では使いにくいが、見通しが立てやすい
    • 3F: 平面/空間の判別に注意
      • |x − a| = r → 平面なら円、空間なら球面
      • (x − a)·(x − b) = 0 → 平面なら円、空間なら球面
      • (x − a)·n = 0 → 平面なら直線、空間なら平面
      • 💡 「答えを円周にしないで球面に」
