  • ⑤ 直線・平面の方程式（空間中心）
    • 末尾: 「直線の方程式を求めよ」／「平面の方程式を求めよ」／「点と平面の距離」
    • 5A: 直線のベクトル方程式
      • 点A(a)を通り、方向ベクトル l に平行: x = a + tl
      • 点A(a,b,c)を通り、方向(α,β,γ)に平行:
      • (x−a)/α = (y−b)/β = (z−c)/γ
    • 5B: 平面のベクトル方程式
      • 3点A,B,Cを通る: x = αa + βb + γc, α+β+γ=1
      • 点Aを通り u, v で張られる: x = a + su + tv
      • 点Aを通り、法線nに平行: (x−a)·n = 0
      • 成分形: α(x−a) + β(y−b) + γ(z−c) = 0
    • 5C: 法線ベクトルを求める
      • 答案用: n = (x,y,z) と置き、n·u=0, n·v=0 を連立
      • 裏計算: 外積 u × v を計算
      • u×v = (u₂v₃−u₃v₂, u₃v₁−u₁v₃, u₁v₂−u₂v₁)
      • 💡 答案では外積を使わず内積=0で書く
    • 5D: 直線と平面の交点
      • 直線をパラメータ表示 x = a + tl
      • 平面の方程式に代入
      • t を解く → 交点
    • 5E: 2直線の交点／ねじれの判定
      • 各直線をパラメータ表示
      • 連立して解の有無を見る
      • 解なし → ねじれの位置
    • 5F: 点と直線の距離
      • 点から直線へ垂線の足 H を出す（4-A 正射影）
      • 距離 = |PH|
    • 5G: 点と平面の距離
      • 公式: d = |α(x₀−a) + β(y₀−b) + γ(z₀−c)| / √(α²+β²+γ²)
      • 別解: 平面上の点AをとりAPのn方向への正射影の長さ
    • 5H: 共通垂線・最短距離
      • 各直線上に動点 P, Q を置く
      • PQ·l₁ = 0, PQ·l₂ = 0 を連立
      • 最短距離 |PQ|
