  • ⑥ 共線・共面・平行・垂直の判定/証明
    • 末尾: 「同一直線上にあることを示せ」／「同一平面上にあることを示せ」／「垂直であることを示せ」
    • 6A: 3点が同一直線上
      • 条件: AB = k·AC を満たす実数 k が存在
      • ⇔ OC = tOA + (1−t)OB
      • 2つのベクトルを書く
      • 平行関係（実数倍）を示す
    • 6B: 4点が同一平面上
      • OD = sOA + tOB + uOC, s+t+u=1
      • ⇔ AD = xAB + yAC（A始点）
      • 3つのベクトルを取る
      • 一方を他2つの一次結合で表す
    • 6C: 平行条件・垂直条件（成分）
      • 平面: (a,b)//(c,d) ⇔ ad − bc = 0
      • : (a,b)⊥(c,d) ⇔ ac + bd = 0
      • 空間: u // v ⇔ u = kv
      • : u ⊥ v ⇔ u·v = 0
    • 6D: 直線が平面に直交を示す
      • 平面の基底 u, v を取る
      • l·u = 0 かつ l·v = 0 を示す
    • 6E: 一次独立性の活用
      • 同じベクトルを2通りに表したとき
      • xa + yb + zc = x'a + y'b + z'c
      • ⇒ a,b,c一次独立なら x=x', y=y', z=z'
      • 💡 係数比較の根拠。連立を解く強力な道具
    • 6F: 対称性ある条件の式変形証明
      • 例: AP·BP + BP·CP + CP·AP = 0
      • 始点を統一（位置ベクトル化）
      • 平方完成、または |...|² = ... の形に
      • 図形的解釈（球面・平面・円）
