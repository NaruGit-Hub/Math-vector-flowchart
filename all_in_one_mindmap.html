ベクトル：定石フローチャート

難易度： 基本 標準 応用 発展 ｜ ▶ をタップで展開
  • ベクトル：問題文末尾 → フロー判定
    • 方針の上位フロー（解析の前にやること）
      • Step 0: 図を描く（鉄則）
      • Step 1: 幾何的考察できないか？（相似・直角・正弦定理・余弦定理）
      • Step 2: 解析的考察に進む（位置ベクトル設定 or 座標設定）
      • 始点をどこに？ → 道具1
      • 位置ベクトル or 座標 → 道具7・道具8
    • 末尾キーワード → フロー
      • 「長さ・面積・体積・なす角」 → ① 図形量を求める
      • 「位置ベクトル・分点・OPを表せ・交点」 → ② 点の位置・分点を求める
      • 「動く範囲・存在領域を図示」 → ③ 終点範囲・軌跡を求める
      • 「垂線の足・正射影・外心」 → ④ 垂線・正射影・外心
      • 「直線/平面の方程式・距離」 → ⑤ 直線・平面の方程式（空間中心）
      • 「同一直線/平面・垂直を示せ」 → ⑥ 共線・共面・平行・垂直の判定/証明
      • 「式の表す図形を答えよ」 → ⑦ ベクトル方程式の表す図形
  • ① 図形量を求める
    • 末尾: 「長さを求めよ」／「面積を求めよ」／「体積を求めよ」／「なす角を求めよ」
    • 1A: 長さ |AB|, |OP| を求める
      • AB = b − a などベクトルで表す
      • |AB|² = (b−a)·(b−a) = |b|² − 2a·b + |a|²
      • 平方根をとる
      • 💡 「絶対値を見たら2乗 → 内積に結ぶ」が定石
    • 1B: 三角形の面積
      • 2辺ベクトル u, v を決める
      • S = (1/2)√(|u|²|v|² − (u·v)²)
      • 💡 平面なら |x₁y₂ − x₂y₁|/2 でも可。空間ではこの公式が必須。
    • 1C: 四面体の体積（ベクトル）
      • 底面 △OAB の面積 S を求める
      • 平面OABへの c の正射影を引いて高さ h
      • V = (1/3) × S × h
      • 💡 裏計算: V = (1/6)|a·(b×c)|（外積は答案不可）
    • 1D: 平行六面体の体積
      • 底面の平行四辺形の面積 S
      • 高さ h（法線方向への正射影）
      • V = S × h
      • 💡 裏計算: V = |a·(b×c)|
    • 1E: 2ベクトルのなす角
      • cosθ = (a·b) / (|a||b|)
      • 範囲 0 ≦ θ ≦ π に注意
      • 💡 直線のなす角(0〜π/2)とベクトルのなす角(0〜π)は別物
    • 1F: 2直線のなす角
      • 方向ベクトルを取り、内積でcos
      • 鋭角に直す（必要なら絶対値）
    • 1G: 平面と直線・2平面のなす角
      • 法線ベクトルを取る
      • 内積で計算
      • 「90°−（法線同士のなす角）」等、向きを補正
  • ② 点の位置・分点を求める
    • 末尾: 「位置ベクトルを表せ」／「OPを表せ」／「s:tに内分する点を求めよ」／「交点を求めよ」
    • 2A: 内分点・外分点の公式
      • m:n に内分: p = (na + mb) / (m+n)
      • m:n に外分: p = (−na + mb) / (m−n)
      • 「m:n に外分」⇔「m:(−n) に内分」として覚える
    • 2B: 共線条件で点を決定
      • OP = (1−t)OA + t·OB と置く
      • 別の条件（直線CD上にある等）と連立
      • 一次独立性より係数比較で t を決定
      • 💡 「Pは直線AB上」+「別の条件」という形を見たら
    • 2C: 共面条件で点を決定（空間）
      • OP = sOA + tOB + uOC, s+t+u=1 と置く
      • 別の条件と連立
      • 係数比較
    • 2D: 角の二等分線との交点
      • ∠AOBの二等分線方向は a/|a| + b/|b|
      • AP:PB = OA:OB を使って OP = (|b|a + |a|b)/(|a|+|b|)
      • 💡 ひし形の公式 OP // a/|a| + b/|b| でも可
    • 2E: 三角形・四面体の重心
      • 三角形 ABC の重心: g = (a+b+c)/3
      • 四面体 ABCD の重心: g = (a+b+c+d)/4
      • 💡 GA + GB + GC = 0（始点を重心に取れば成立）
    • 2F: 内心の位置ベクトル
      • 角の二等分線の性質から BD:DC = c:b
      • D の位置ベクトルを出す
      • AI:ID = (b+c):a を再利用
      • 結果: i = (a·a + b·b + c·c)/(a+b+c)
      • 💡 a, b, c は対辺の長さ（覚える価値あり）
    • 2G: 外心の位置ベクトル
      • OR = xa + yb と置く
      • 辺OA, OBの中点を M, N
      • RM·OA = 0, RN·OB = 0 を連立
      • x, y を解く
      • 💡 外心を始点に取れる場合は |a|=|b|=|c|=R が使えてラク
    • 2H: 「sa + tb + uc = 0」型の利用
      • パターン1: 一文字消去 a = −(tb+uc)/s
      • パターン2: 内積を見る a·(sa+tb+uc) = 0
      • パターン3: 1つを移項して2乗 |sa|² = |−(tb+uc)|²
      • 💡 3ベクトルの和が0なら2乗が定石
  • ③ 終点範囲・軌跡を求める
    • 末尾: 「Pの動く範囲を図示せよ」／「Pの存在領域を求めよ」／「領域を求めよ」
    • 3A: 基本パターン（暗記）
      • OP = sa + tb のとき
      • ・s, t 実数全体 → 全平面
      • ・s+t = 1 → 直線AB
      • ・s+t = 1, s≧0,t≧0 → 線分AB
      • ・s+t ≦ 1, s≧0,t≧0 → △OAB周+内部
      • 空間も同様: s+t+u に拡張
    • 3A2: 基本パターン（空間）
      • OP = la + mb + nc のとき
      • ・l+m+n = 1 → 平面ABC
      • ・l+m+n = 1, 各≧0 → △ABC周+内部
      • ・l+m+n ≦ 1, 各≧0 → 四面体OABC周+内部
    • 3B: 一般の終点範囲（3手順型）
      • s（または t）を固定
      • 動く方の変数で線を作る
      • 固定した変数を動かして面に拡大
      • 💡 0≦a≦1, 1/2≦b≦1 のような長方形領域で使う
    • 3C: (s+t) を作って3-Aに帰着
      • OP = (s+t) × {(sa+tb)/(s+t)} と変形
      • 内側の点 Q は線分AB上を動く
      • P は OQ を (s+t) 倍した点 → 図示
      • 💡 「パラメータについて整理する」の発想
    • 3D: 始点を変えて簡単化
      • a, b, c の係数和が a+b+c=1 等のとき
      • 始点を A に統一（→AP, AB, AC で書き直す）
      • 制約条件も書き換え
      • 3-A に帰着
    • 3E: 斜交座標による解法
      • a, b 方向を斜交軸とみなす
      • (s, t) 平面で不等式を図示
      • 元の図に戻す
      • 💡 答案では使いにくいが、見通しが立てやすい
    • 3F: 平面/空間の判別に注意
      • |x − a| = r → 平面なら円、空間なら球面
      • (x − a)·(x − b) = 0 → 平面なら円、空間なら球面
      • (x − a)·n = 0 → 平面なら直線、空間なら平面
      • 💡 「答えを円周にしないで球面に」
  • ④ 垂線・正射影・外心
    • 末尾: 「垂線の足を求めよ」／「正射影ベクトルを求めよ」／「外心を求めよ」
    • 4A: 直線（ベクトル）への正射影
      • 公式: OH = (a·b / |a|²) × a
      • 内積 a·b を計算
      • |a|² を計算
      • 公式に代入
      • 💡 「ある点から直線に垂線を下ろす」を見たら即この公式
    • 4B: 垂線の足を交点として求める
      • H = OA + t·AB と置く（直線AB上）
      • PH·AB = 0（垂直条件）
      • t を決定
      • 💡 公式を使わない別解。連立で解ける
    • 4C: 平面への正射影（空間）
      • 公式は「ベクトル→ベクトル」なので、まず法線nに正射影
      • n への正射影 = (n·x/|n|²)·n
      • 平面αへの正射影 = x − (n·x/|n|²)·n
    • 4D: 三角形の外心
      • OR = xa + yb と置く
      • 辺の中点 M, N を取る
      • RM·OA = 0, RN·OB = 0 を連立
      • x, y を決定
      • 💡 外心は「垂直二等分線の交点」
    • 4E: 正射影 × 外心の合わせ技
      • 外心から辺へ垂線を下ろすと中点に落ちる
      • 外心と垂直二等分線の相性は◎
    • 4F: 平行六面体の高さ
      • 底面の法線方向 n
      • 高さ |OH| = |b·n| / |n|
      • 体積 V = 底面積 × 高さ
  • ⑤ 直線・平面の方程式（空間中心）
    • 末尾: 「直線の方程式を求めよ」／「平面の方程式を求めよ」／「点と平面の距離」
    • 5A: 直線のベクトル方程式
      • 点A(a)を通り、方向ベクトル l に平行: x = a + tl
      • 点A(a,b,c)を通り、方向(α,β,γ)に平行:
      • (x−a)/α = (y−b)/β = (z−c)/γ
    • 5B: 平面のベクトル方程式
      • 3点A,B,Cを通る: x = αa + βb + γc, α+β+γ=1
      • 点Aを通り u, v で張られる: x = a + su + tv
      • 点Aを通り、法線nに平行: (x−a)·n = 0
      • 成分形: α(x−a) + β(y−b) + γ(z−c) = 0
    • 5C: 法線ベクトルを求める
      • 答案用: n = (x,y,z) と置き、n·u=0, n·v=0 を連立
      • 裏計算: 外積 u × v を計算
      • u×v = (u₂v₃−u₃v₂, u₃v₁−u₁v₃, u₁v₂−u₂v₁)
      • 💡 答案では外積を使わず内積=0で書く
    • 5D: 直線と平面の交点
      • 直線をパラメータ表示 x = a + tl
      • 平面の方程式に代入
      • t を解く → 交点
    • 5E: 2直線の交点／ねじれの判定
      • 各直線をパラメータ表示
      • 連立して解の有無を見る
      • 解なし → ねじれの位置
    • 5F: 点と直線の距離
      • 点から直線へ垂線の足 H を出す（4-A 正射影）
      • 距離 = |PH|
    • 5G: 点と平面の距離
      • 公式: d = |α(x₀−a) + β(y₀−b) + γ(z₀−c)| / √(α²+β²+γ²)
      • 別解: 平面上の点AをとりAPのn方向への正射影の長さ
    • 5H: 共通垂線・最短距離
      • 各直線上に動点 P, Q を置く
      • PQ·l₁ = 0, PQ·l₂ = 0 を連立
      • 最短距離 |PQ|
  • ⑥ 共線・共面・平行・垂直の判定/証明
    • 末尾: 「同一直線上にあることを示せ」／「同一平面上にあることを示せ」／「垂直であることを示せ」
    • 6A: 3点が同一直線上
      • 条件: AB = k·AC を満たす実数 k が存在
      • ⇔ OC = tOA + (1−t)OB
      • 2つのベクトルを書く
      • 平行関係（実数倍）を示す
    • 6B: 4点が同一平面上
      • OD = sOA + tOB + uOC, s+t+u=1
      • ⇔ AD = xAB + yAC（A始点）
      • 3つのベクトルを取る
      • 一方を他2つの一次結合で表す
    • 6C: 平行条件・垂直条件（成分）
      • 平面: (a,b)//(c,d) ⇔ ad − bc = 0
      • : (a,b)⊥(c,d) ⇔ ac + bd = 0
      • 空間: u // v ⇔ u = kv
      • : u ⊥ v ⇔ u·v = 0
    • 6D: 直線が平面に直交を示す
      • 平面の基底 u, v を取る
      • l·u = 0 かつ l·v = 0 を示す
    • 6E: 一次独立性の活用
      • 同じベクトルを2通りに表したとき
      • xa + yb + zc = x'a + y'b + z'c
      • ⇒ a,b,c一次独立なら x=x', y=y', z=z'
      • 💡 係数比較の根拠。連立を解く強力な道具
    • 6F: 対称性ある条件の式変形証明
      • 例: AP·BP + BP·CP + CP·AP = 0
      • 始点を統一（位置ベクトル化）
      • 平方完成、または |...|² = ... の形に
      • 図形的解釈（球面・平面・円）
  • ⑦ ベクトル方程式の表す図形
    • 末尾: 「式が表す図形を答えよ」／「存在範囲を求めよ」
    • 7A: パラメータ表示
      • 形: p = sa + tb
      • ・パラメータ1つ → 直線
      • ・パラメータ2つ → 平面（空間内）
    • 7B: 内積=一次の形
      • 形: a·(p − q) = 0
      • ・平面: 点Qを通り、aに垂直な直線
      • ・空間: 点Qを通り、aに垂直な平面
      • 💡 内積形の1次式 → 直線/平面
    • 7C: 絶対値=定数
      • 形: |x − a| = r
      • ・平面: 中心A、半径rの円
      • ・空間: 中心A、半径rの球面
    • 7D: 内積形の積=0
      • 形: (x − a)·(x − b) = 0
      • ・平面: ABを直径とする円
      • ・空間: ABを直径とする球面
    • 7E: 一般の2次形（平方完成）
      • 形: |p|² + a·p + b = 0
      • 平方完成して |p − c|² = R² の形に
      • 円（平面）/ 球面（空間）と判定
      • 💡 2次の項を含むなら円・球面が予想される
    • 7F: 因数分解で2平面の和に
      • 2次式が因数分解できる場合
      • 因数分解 → 2つの1次式の積
      • 「内積=0 または 内積=0」の和事象
    • 7G: 空間内の円
      • 定石1: 「球面 かつ 平面」と捉える
      • |x−a|=r かつ (x−b)·n = 0
      • 定石2: パラメータ表示
      • x = c + (r cosθ)p + (r sinθ)q
    • 7H: 平面/空間の判定に注意
      • 同じ式でも「平面」か「空間」かで答えが変わる
      • ・平面 → 円・直線
      • ・空間 → 球面・平面
      • 「答えを円周にしないで球面に」
  • 道具集（フロー横断）
    • 始点統一の判断
      • 条件が対称（A,B,C等価） → 位置ベクトル（始点はどこでも）
      • 1点だけ条件が違う → その点を始点に統一
      • 外心が絡む → 外心を始点（|a|=|b|=|c|=R が使える）
    • 一次独立／一次結合
      • 平面: a, b 一次独立 ⇔ sa + tb = 0 ⇒ s = t = 0
      • 任意のベクトル x = pa + qb で一意に表せる
      • 係数比較ができるのが最大の武器
    • 絶対値の2乗 → 内積
      • |a|² = a·a
      • |a ± b|² = |a|² ± 2a·b + |b|²
      • 「絶対値を見たら2乗」が定石
    • 角の二等分線
      • ∠AOBの二等分線方向: a/|a| + b/|b|
      • 向きに注意（外角と区別）
    • 重心の位置ベクトル
      • 三角形 ABC の重心: g = (a+b+c)/3
      • 四面体 ABCD の重心: g = (a+b+c+d)/4
      • GA + GB + GC = 0（始点を重心に取れば）
    • 外積（裏計算用・答案不可）
      • 定義: a×b = (a₂b₃−a₃b₂, a₃b₁−a₁b₃, a₁b₂−a₂b₁)
      • 性質: a×b は a, b の両方に垂直
      • |a×b| = a, b が作る平行四辺形の面積
      • 用途: 法線ベクトルの「答え合わせ」
      • 平行六面体の体積 V = |a·(b×c)|
      • 四面体の体積 = V/6
      • ⚠ 答案では「内積=0を満たす」と書く
    • 対称性のあるベクトルの座標設定
      • 平面: 長さ1の a, b → a=(1,0), b=(cosθ, sinθ)
      • 空間: 長さ1の a, b, c → a=(1,0,0), b=(cosθ, sinθ, 0),
      • c=(l, m, n) [l²+m²+n²=1]
      • 設定する文字を最小化するのがコツ
    • 平面 vs 空間の解法選択
      • ベクトルで解く（特に四面体・平行六面体で有効）
      • 座標設定で解く（直交軸が見えるとき）
      • 断面図など幾何的に解く（対称面・対称軸を含む面）
      • 展開図を考える（最短経路など）
    • ベクトル和 sa+tb+uc=0 の利用
      • 一文字消去: a = −(tb+uc)/s
      • 内積を見る: a·(sa+tb+uc) = 0
      • 対称性を見る
      • 1つ移項して2乗 |sa|² = |−(tb+uc)|²
