# 数学フローチャート — ベクトル

高校数学（数B・数C）「ベクトル」を、問題文の末尾キーワードから機械的にたどれるフローチャート資料です。

## 構成

- `index.html` — 入口ページ（使い方ガイド付き）
- `all_in_one_mindmap.html` — 全フロー統合のマインドマップ版（推奨）
- `all_in_one_text.html` — 全フロー統合のテキスト版（印刷向き）
- `flow1_*.html` 〜 `flow7_*.html` — 個別フロー（テキスト版／マインドマップ版）
- `images/` — 図（SVG）

## 使い方ガイド

図形問題の解法は複数あります。この資料は **「ベクトルで解くと決めたあと」** の定石を整理したものです。

- Step 0: 図を描く
- Step 1: 幾何的考察で解けないか試す（相似・直角・正弦定理・余弦定理 等）
- Step 2: 解析的考察に進む。3つの選択肢：**ベクトル**（この資料）／座標設定／複素数平面
- Step 3: ベクトルで解くと決めたら、末尾キーワードから該当フローへ

## フロー一覧

| # | タイトル | 末尾キーワード例 |
|---|---|---|
| ① | 図形量を求める | 長さ・面積・体積・なす角 |
| ② | 点の位置・分点 | 位置ベクトル・OPを表せ・交点 |
| ③ | 終点範囲・軌跡 | 動く範囲・存在領域を図示 |
| ④ | 垂線・正射影・外心 | 垂線の足・正射影・外心 |
| ⑤ | 直線・平面の方程式 | 直線/平面の方程式・距離 |
| ⑥ | 共線・共面・平行・垂直の判定/証明 | 同一直線/平面・垂直を示せ |
| ⑦ | ベクトル方程式の表す図形 | 式の表す図形を答えよ |

別添：道具集（始点統一・一次独立・絶対値の2乗→内積・角の二等分線・重心・外積（裏計算用）・座標設定 等）

## 難易度の色分け

- ⬛ 基本（黒）
- 🟧 標準（黄）
- 🟦 応用（青）
- 🟥 発展（赤）

## 技術

- HTML/CSS/JS のみ
- 数式表示は MathJax 3 (CDN: jsdelivr) を使用
- 図は SVG で埋め込み
- スマホ対応
- 初回読み込み後はブラウザキャッシュにより高速動作

## 注意事項

数式表示にはネット接続が必要です（初回のみ。以降はキャッシュで高速）。
完全オフラインでの数式表示が必要な場合は MathJax のローカル同梱が必要です。
