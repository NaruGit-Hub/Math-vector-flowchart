② 点の位置・分点を求める標準

末尾キーワード: 「位置ベクトルを表せ」／「OPを表せ」／「s:tに内分する点を求めよ」／「交点を求めよ」

問題文の末尾が「位置ベクトルを表せ／分点を求めよ」のとき。共線・共面条件と一次独立性で連立する。

2A: 内分点・外分点の公式基本

  • m:n に内分: p = (na + mb) / (m+n)
  • m:n に外分: p = (−na + mb) / (m−n)
  • 「m:n に外分」⇔「m:(−n) に内分」として覚える

2B: 共線条件で点を決定標準

  • OP = (1−t)OA + t·OB と置く
  • 別の条件（直線CD上にある等）と連立
  • 一次独立性より係数比較で t を決定
「Pは直線AB上」+「別の条件」という形を見たら

2C: 共面条件で点を決定（空間）標準

  • OP = sOA + tOB + uOC, s+t+u=1 と置く
  • 別の条件と連立
  • 係数比較

2D: 角の二等分線との交点標準

  • ∠AOBの二等分線方向は a/|a| + b/|b|
  • AP:PB = OA:OB を使って OP = (|b|a + |a|b)/(|a|+|b|)
ひし形の公式 OP // a/|a| + b/|b| でも可

2E: 三角形・四面体の重心基本

  • 三角形 ABC の重心: g = (a+b+c)/3
  • 四面体 ABCD の重心: g = (a+b+c+d)/4
GA + GB + GC = 0（始点を重心に取れば成立）

2F: 内心の位置ベクトル応用

  • 角の二等分線の性質から BD:DC = c:b
  • D の位置ベクトルを出す
  • AI:ID = (b+c):a を再利用
  • 結果: i = (a·a + b·b + c·c)/(a+b+c)
a, b, c は対辺の長さ（覚える価値あり）

2G: 外心の位置ベクトル応用

  • OR = xa + yb と置く
  • 辺OA, OBの中点を M, N
  • RM·OA = 0, RN·OB = 0 を連立
  • x, y を解く
外心を始点に取れる場合は |a|=|b|=|c|=R が使えてラク

2H: 「sa + tb + uc = 0」型の利用応用

  • パターン1: 一文字消去 a = −(tb+uc)/s
  • パターン2: 内積を見る a·(sa+tb+uc) = 0
  • パターン3: 1つを移項して2乗 |sa|² = |−(tb+uc)|²
3ベクトルの和が0なら2乗が定石
