この資料の使い方

図形問題に出会ったとき、解法の選択肢は複数あります。この資料は「ベクトルで解くと決めたあと」の定石を整理したものです。

Step 0: まず図を描く
図形問題を解く前提として、必ず図を描く。
Step 1: 幾何的考察で解けないか試す
相似・合同・直角・円周角・接弦定理・方べきの定理・正弦定理・余弦定理 など、初等幾何の道具で解ければ計算量が大幅に減る。まずこれを試す。
Step 2: 解析的考察に進む（3つの選択肢）
幾何で解けない／面倒な場合、以下から選ぶ：
  • ベクトル：対称性のある図形・四面体・平行六面体・空間問題で強い。← この資料はこれ専用
  • 座標設定：直交軸・原点が自然に取れるとき。計算量を減らしやすい。
  • 複素数平面：回転・拡大縮小が絡む問題で強い（数Cで学習）。
Step 3: ベクトルで解くと決めたら
問題文の末尾キーワードを見て、下の早見表から該当するフローへ進む。

① 図形量を求める標準

末尾キーワード: 「長さを求めよ」／「面積を求めよ」／「体積を求めよ」／「なす角を求めよ」

問題文の末尾が「長さ・面積・体積・なす角」のとき、ベクトルから具体的な数値を出すフロー。

1A: 長さ \(|\vec{AB}|,\ |\vec{OP}|\) を求める基本

  • \(\vec{AB} = \vec{b} - \vec{a}\) などベクトルで表す
  • \(|\vec{AB}|^2 = (\vec{b}-\vec{a})\cdot(\vec{b}-\vec{a}) = |\vec{b}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{a}|^2\)
  • 平方根をとる
「絶対値を見たら2乗 → 内積に結ぶ」が定石
📝 例題
問題三角形 OAB において、\(|\vec{a}|=3,\ |\vec{b}|=2,\ \vec{a}\cdot\vec{b}=4\) とする。辺 AB の長さを求めよ。
方針\(\vec{AB}=\vec{b}-\vec{a}\) と書き、絶対値の2乗→内積の定石を使う。
解答
  • \(\vec{AB} = \vec{b} - \vec{a}\) より、
  • \(|\vec{AB}|^2 = |\vec{b}-\vec{a}|^2 = |\vec{b}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{a}|^2\)
  • \(\quad = 4 - 8 + 9 = 5\)
  • よって \(|\vec{AB}| = \sqrt{5}\)
答え\(|\vec{AB}| = \sqrt{5}\)

1B: 三角形の面積標準

  • 2辺ベクトル \(\vec{u},\ \vec{v}\) を決める
  • \(\displaystyle S = \frac{1}{2}\sqrt{|\vec{u}|^2|\vec{v}|^2 - (\vec{u}\cdot\vec{v})^2}\)
平面なら \(\dfrac{|x_1 y_2 - x_2 y_1|}{2}\) でも可。空間ではこの公式が必須。
📝 例題
問題\(|\vec{a}|=2,\ |\vec{b}|=3,\ \vec{a}\cdot\vec{b}=4\) のとき、\(\vec{a},\vec{b}\) で張られる三角形 OAB の面積 \(S\) を求めよ。
方針公式 \(\displaystyle S = \frac{1}{2}\sqrt{|\vec{a}|^2|\vec{b}|^2 - (\vec{a}\cdot\vec{b})^2}\) に代入するだけ。
解答
  • \(\displaystyle S = \frac{1}{2}\sqrt{|\vec{a}|^2|\vec{b}|^2 - (\vec{a}\cdot\vec{b})^2}\)
  • \(\displaystyle \quad = \frac{1}{2}\sqrt{4 \cdot 9 - 16} = \frac{1}{2}\sqrt{20} = \sqrt{5}\)
答え\(S = \sqrt{5}\)

1C: 四面体の体積（ベクトル）応用

  • 底面 \(\triangle OAB\) の面積 \(S\) を求める
  • 平面 \(OAB\) への \(\vec{c}\) の正射影を引いて高さ \(h\)
  • \(\displaystyle V = \frac{1}{3} \times S \times h\)
裏計算: \(\displaystyle V = \frac{1}{6}|\vec{a}\cdot(\vec{b}\times\vec{c})|\)（外積は答案不可）
📝 例題
問題四面体 OABC において、\(|\vec{a}|=|\vec{b}|=|\vec{c}|=1\)、\(\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{c}=\vec{c}\cdot\vec{a}=\dfrac{1}{2}\) のとき、四面体の体積を求めよ。
方針底面の面積 → 高さ（正射影で求める）→ 体積、の順。対称性が高いので計算は楽。
解答
  • 底面 \(\triangle OAB\) の面積：\(\displaystyle S = \frac{1}{2}\sqrt{1\cdot 1 - \left(\frac{1}{2}\right)^2} = \frac{\sqrt{3}}{4}\)
  • 高さ \(h\)：\(\vec{c}\) の平面 \(OAB\) への正射影を求める。
  • 対称性より \(\vec{c}\) を \(\vec{a},\vec{b}\) に分解して \(\vec{c} = s\vec{a}+t\vec{b}+\vec{h}\) と置くと、
  • \(\vec{h}\cdot\vec{a}=0,\ \vec{h}\cdot\vec{b}=0\) より \(\displaystyle s = t = \frac{1}{3}\)
  • \(\displaystyle |\vec{h}|^2 = \left|\vec{c}-\frac{1}{3}\vec{a}-\frac{1}{3}\vec{b}\right|^2 = \frac{2}{3}\)
  • よって \(\displaystyle h = \sqrt{\frac{2}{3}}\)
  • \(\displaystyle V = \frac{1}{3} \cdot \frac{\sqrt{3}}{4} \cdot \sqrt{\frac{2}{3}} = \frac{\sqrt{2}}{12}\)
答え\(\displaystyle V = \frac{\sqrt{2}}{12}\)

1D: 平行六面体の体積応用

  • 底面の平行四辺形の面積 \(S\)
  • 高さ \(h\)（法線方向への正射影）
  • \(V = S \times h\)
裏計算: \(V = |\vec{a}\cdot(\vec{b}\times\vec{c})|\)
📝 例題
問題3つのベクトル \(\vec{a},\vec{b},\vec{c}\) で張られる平行六面体について、\(|\vec{a}|=|\vec{b}|=|\vec{c}|=1\)、互いのなす角がすべて \(60°\) のとき、体積 \(V\) を求めよ。
方針裏計算で \(V=|\vec{a}\cdot(\vec{b}\times\vec{c})|\) を予想し、答案では「底面積×高さ」で書く。
解答
  • 底面 \(\vec{a},\vec{b}\) の作る平行四辺形の面積：\(\displaystyle S = \sqrt{1\cdot 1 - \frac{1}{4}} = \frac{\sqrt{3}}{2}\)
  • 高さ：\(\vec{c}\) を底面に正射影した残りの長さ。
  • \(\vec{c} = s\vec{a}+t\vec{b}+\vec{h}\) と置き、内積条件から \(\displaystyle s = t = \frac{1}{3}\)
  • \(\displaystyle h = |\vec{h}| = \sqrt{\frac{2}{3}}\)
  • \(\displaystyle V = S \cdot h = \frac{\sqrt{3}}{2} \cdot \sqrt{\frac{2}{3}} = \frac{\sqrt{2}}{2}\)
答え\(\displaystyle V = \frac{\sqrt{2}}{2}\)

1E: 2ベクトルのなす角基本

  • \(\displaystyle \cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}\)
  • 範囲 \(0 \leqq \theta \leqq \pi\) に注意
直線のなす角(\(0\sim\pi/2\))とベクトルのなす角(\(0\sim\pi\))は別物
📝 例題
問題\(|\vec{a}|=2,\ |\vec{b}|=3,\ \vec{a}\cdot\vec{b}=-3\) のとき、\(\vec{a}\) と \(\vec{b}\) のなす角 \(\theta\) を求めよ。
方針公式 \(\displaystyle \cos\theta = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}\) に代入。
解答
  • \(\displaystyle \cos\theta = \frac{-3}{2\cdot 3} = -\frac{1}{2}\)
  • \(0 \leqq \theta \leqq \pi\) より、
答え\(\displaystyle \theta = \frac{2}{3}\pi\)

1F: 2直線のなす角標準

  • 方向ベクトルを取り、内積でcos
  • 鋭角に直す（必要なら絶対値）
📝 例題
問題2直線 \(\ell_1\): 方向ベクトル \(\vec{u}=(1,2,2)\)、\(\ell_2\): 方向ベクトル \(\vec{v}=(2,-1,2)\) のなす角を求めよ。
方針方向ベクトルの内積でcosを出し、絶対値で鋭角に直す。
解答
  • \(\vec{u}\cdot\vec{v} = 2 - 2 + 4 = 4\)
  • \(|\vec{u}| = 3,\ |\vec{v}| = 3\)
  • \(\displaystyle \cos\theta = \frac{|4|}{3\cdot 3} = \frac{4}{9}\)（鋭角に直すため絶対値）
答え\(\displaystyle \theta = \arccos\frac{4}{9}\)

1G: 平面と直線・2平面のなす角発展

  • 法線ベクトルを取る
  • 内積で計算
  • 「\(90°\)−（法線同士のなす角）」等、向きを補正
📝 例題
問題平面 \(\alpha\): \(x+y+z=1\) と直線 \(\ell\): 方向ベクトル \(\vec{l}=(1,1,0)\) のなす角を求めよ。
方針平面の法線 \(\vec{n}\) と \(\vec{l}\) のなす角 \(\varphi\) を出し、\(90°-\varphi\) が直線と平面のなす角。
解答
  • 法線 \(\vec{n}=(1,1,1)\)
  • \(\vec{l}\cdot\vec{n} = 2,\ |\vec{l}|=\sqrt{2},\ |\vec{n}|=\sqrt{3}\)
  • \(\displaystyle \cos\varphi = \frac{2}{\sqrt{2}\cdot\sqrt{3}} = \frac{\sqrt{6}}{3}\)（鋭角）
  • よって直線と平面のなす角 \(\theta = 90° - \varphi\) より
  • \(\displaystyle \sin\theta = \cos\varphi = \frac{\sqrt{6}}{3}\)
答え\(\displaystyle \sin\theta = \frac{\sqrt{6}}{3}\)（または \(\theta = \arcsin\frac{\sqrt{6}}{3}\)）

② 点の位置・分点を求める標準

末尾キーワード: 「位置ベクトルを表せ」／「OPを表せ」／「s:tに内分する点を求めよ」／「交点を求めよ」

問題文の末尾が「位置ベクトルを表せ／分点を求めよ」のとき。共線・共面条件と一次独立性で連立する。

2A: 内分点・外分点の公式基本

  • \(m:n\) に内分: \(\displaystyle \vec{p} = \frac{n\vec{a} + m\vec{b}}{m+n}\)
  • \(m:n\) に外分: \(\displaystyle \vec{q} = \frac{-n\vec{a} + m\vec{b}}{m-n}\)
  • 「\(m:n\) に外分」⇔「\(m:(-n)\) に内分」として覚える
📝 例題
問題線分 AB を \(2:3\) に内分する点 P と、\(2:3\) に外分する点 Q の位置ベクトルを、\(\vec{a}=\vec{OA},\ \vec{b}=\vec{OB}\) で表せ。
方針公式に代入。外分は内分の式の符号を1か所変えれば良い。
解答
  • 内分: \(\displaystyle \vec{OP} = \frac{3\vec{a} + 2\vec{b}}{2+3} = \frac{3\vec{a}+2\vec{b}}{5}\)
  • 外分: \(\displaystyle \vec{OQ} = \frac{-3\vec{a} + 2\vec{b}}{2-3} = 3\vec{a} - 2\vec{b}\)
答え\(\displaystyle \vec{OP} = \frac{3\vec{a}+2\vec{b}}{5},\ \vec{OQ} = 3\vec{a}-2\vec{b}\)

2B: 共線条件で点を決定標準

  • \(\vec{OP} = (1-t)\vec{OA} + t\vec{OB}\) と置く
  • 別の条件（直線CD上にある等）と連立
  • 一次独立性より係数比較で \(t\) を決定
「P は直線AB上」+「別の条件」という形を見たら
📝 例題
問題三角形 OAB において、辺 OA を \(1:2\) に内分する点を M、辺 OB の中点を N とする。線分 AN と BM の交点を P とするとき、\(\vec{OP}\) を \(\vec{a}=\vec{OA},\ \vec{b}=\vec{OB}\) で表せ。
方針「P は直線AN上」かつ「P は直線BM上」を立式して連立。係数比較で決定。
解答
  • P が直線 AN 上：\(\vec{OP} = (1-s)\vec{a} + s\cdot\dfrac{\vec{b}}{2}\)
  • P が直線 BM 上：\(\vec{OP} = t\cdot\dfrac{\vec{a}}{3} + (1-t)\vec{b}\)
  • \(\vec{a},\vec{b}\) は一次独立より係数比較：
  • \(\displaystyle 1-s = \frac{t}{3},\ \frac{s}{2} = 1-t\)
  • 連立して \(\displaystyle s = \frac{4}{5},\ t = \frac{3}{5}\)
  • \(\displaystyle \vec{OP} = \frac{1}{5}\vec{a} + \frac{2}{5}\vec{b}\)
答え\(\displaystyle \vec{OP} = \frac{1}{5}\vec{a} + \frac{2}{5}\vec{b}\)

2C: 共面条件で点を決定（空間）標準

  • \(\vec{OP} = s\vec{OA} + t\vec{OB} + u\vec{OC},\ s+t+u=1\) と置く
  • 別の条件と連立
  • 係数比較
📝 例題
問題四面体 OABC において、辺 OA, OB, OC の中点をそれぞれ L, M, N とする。直線 OG（G は \(\triangle ABC\) の重心）と平面 LMN の交点 P について、\(\vec{OP}\) を \(\vec{a},\vec{b},\vec{c}\) で表せ。
方針P が直線 OG 上＝1パラメータ、P が平面 LMN 上＝係数和=1（共面条件）を立式して連立。
解答
  • \(\displaystyle \vec{OG} = \frac{\vec{a}+\vec{b}+\vec{c}}{3}\)
  • P は直線 OG 上：\(\displaystyle \vec{OP} = u\cdot\frac{\vec{a}+\vec{b}+\vec{c}}{3}\)
  • P は平面 LMN 上：\(\vec{OP} = s\vec{OL} + t\vec{OM} + r\vec{ON}\)（\(s+t+r=1\)）
  • \(\displaystyle = \frac{s}{2}\vec{a} + \frac{t}{2}\vec{b} + \frac{r}{2}\vec{c}\)
  • 両式で係数比較：\(\displaystyle \frac{u}{3} = \frac{s}{2} = \frac{t}{2} = \frac{r}{2}\) かつ \(s+t+r=1\)
  • 対称性より \(\displaystyle s=t=r=\frac{1}{3}\)、よって \(\displaystyle \frac{u}{3} = \frac{1}{6}\)、\(\displaystyle u = \frac{1}{2}\)
  • \(\displaystyle \vec{OP} = \frac{1}{6}(\vec{a}+\vec{b}+\vec{c})\)
答え\(\displaystyle \vec{OP} = \frac{1}{6}(\vec{a}+\vec{b}+\vec{c})\)

2D: 角の二等分線との交点標準

  • \(\angle AOB\) の二等分線方向は \(\displaystyle \frac{\vec{a}}{|\vec{a}|} + \frac{\vec{b}}{|\vec{b}|}\)
  • \(AP:PB = OA:OB\) を使って \(\displaystyle \vec{OP} = \frac{|\vec{b}|\vec{a} + |\vec{a}|\vec{b}}{|\vec{a}|+|\vec{b}|}\)
ひし形の公式 \(\vec{OP} \parallel \dfrac{\vec{a}}{|\vec{a}|} + \dfrac{\vec{b}}{|\vec{b}|}\) でも可
📝 例題
問題\(\triangle OAB\) において \(OA=3,\ OB=2\) とする。\(\angle AOB\) の二等分線と辺 AB の交点 P について、\(\vec{OP}\) を \(\vec{a}=\vec{OA},\ \vec{b}=\vec{OB}\) で表せ。
方針\(AP:PB = OA:OB = 3:2\) なので、内分点の公式に代入。
解答
  • \(AP:PB = OA:OB = 3:2\) より P は AB を \(3:2\) に内分。
  • \(\displaystyle \vec{OP} = \frac{2\vec{a} + 3\vec{b}}{3+2} = \frac{2\vec{a}+3\vec{b}}{5}\)
答え\(\displaystyle \vec{OP} = \frac{2\vec{a}+3\vec{b}}{5}\)

2E: 三角形・四面体の重心基本

  • 三角形 ABC の重心: \(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}}{3}\)
  • 四面体 ABCD の重心: \(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}+\vec{d}}{4}\)
\(\vec{GA}+\vec{GB}+\vec{GC}=\vec{0}\)（始点を重心に取れば成立）
📝 例題
問題\(\triangle ABC\) の重心を G、四面体 ABCD の重心を G' とする。それぞれの位置ベクトルを各頂点の位置ベクトルで表せ。
方針公式そのもの。
解答
  • 三角形の重心：\(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}}{3}\)
  • 四面体の重心：\(\displaystyle \vec{g'} = \frac{\vec{a}+\vec{b}+\vec{c}+\vec{d}}{4}\)
答え\(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}}{3},\ \vec{g'} = \frac{\vec{a}+\vec{b}+\vec{c}+\vec{d}}{4}\)

2F: 内心の位置ベクトル応用

  • 角の二等分線の性質から \(BD:DC = c:b\)
  • D の位置ベクトルを出す
  • \(AI:ID = (b+c):a\) を再利用
  • 結果: \(\displaystyle \vec{i} = \frac{a\vec{a} + b\vec{b} + c\vec{c}}{a+b+c}\)
\(a, b, c\) は対辺の長さ（覚える価値あり）
📝 例題
問題\(\triangle ABC\) において \(BC=a,\ CA=b,\ AB=c\) とする。内心 I の位置ベクトル \(\vec{i}\) を、各頂点の位置ベクトルで表せ。
方針角の二等分線を2回使う。BD:DC=c:b、AI:ID=(b+c):a の順で。
解答
  • 角 A の二等分線と辺 BC の交点 D について、\(BD:DC = c:b\)
  • \(\displaystyle \vec{d} = \frac{b\vec{b} + c\vec{c}}{b+c}\)
  • 角 B の二等分線の性質より \(AI:ID = (b+c):a\)
  • \(\displaystyle \vec{i} = \frac{a\vec{a} + (b+c)\vec{d}}{(b+c)+a} = \frac{a\vec{a} + b\vec{b} + c\vec{c}}{a+b+c}\)
答え\(\displaystyle \vec{i} = \frac{a\vec{a} + b\vec{b} + c\vec{c}}{a+b+c}\)

2G: 外心の位置ベクトル応用

  • \(\vec{OR} = x\vec{a} + y\vec{b}\) と置く
  • 辺 OA, OB の中点を M, N
  • \(\vec{RM}\cdot\vec{OA}=0,\ \vec{RN}\cdot\vec{OB}=0\) を連立
  • \(x, y\) を解く
外心を始点に取れる場合は \(|\vec{a}|=|\vec{b}|=|\vec{c}|=R\) が使えてラク
📝 例題
問題\(\triangle OAB\) において \(|\vec{a}|=2,\ |\vec{b}|=3,\ \vec{a}\cdot\vec{b}=4\) とする。外心 R について \(\vec{OR}=x\vec{a}+y\vec{b}\) と置くとき、\(x, y\) を求めよ。
方針外心は OA, OB の垂直二等分線の交点。各中点 M, N について RM⊥OA, RN⊥OB。
解答
  • OA, OB の中点 \(\displaystyle M=\frac{\vec{a}}{2},\ N=\frac{\vec{b}}{2}\)
  • \(\vec{RM}\cdot\vec{a}=0\)：\(\displaystyle \left(\frac{\vec{a}}{2}-x\vec{a}-y\vec{b}\right)\cdot\vec{a} = 0\)
  • \(\displaystyle \frac{|\vec{a}|^2}{2} - x|\vec{a}|^2 - y\vec{a}\cdot\vec{b} = 0\)
  • \(2 - 4x - 4y = 0\) … ①
  • 同様に \(\vec{RN}\cdot\vec{b}=0\) より \(\displaystyle \frac{9}{2} - 4x - 9y = 0\) … ②
  • ①②を解いて \(\displaystyle x = \frac{3}{20},\ y = \frac{7}{20}\)
答え\(\displaystyle x = \frac{3}{20},\ y = \frac{7}{20}\)

2H: 「\(s\vec{a}+t\vec{b}+u\vec{c}=\vec{0}\)」型の利用応用

  • パターン1: 一文字消去 \(\displaystyle \vec{a} = -\frac{t\vec{b}+u\vec{c}}{s}\)
  • パターン2: 内積を見る \(\vec{a}\cdot(s\vec{a}+t\vec{b}+u\vec{c}) = 0\)
  • パターン3: 1つを移項して2乗 \(|s\vec{a}|^2 = |-(t\vec{b}+u\vec{c})|^2\)
3ベクトルの和が0なら2乗が定石
📝 例題
問題\(|\vec{a}|=|\vec{b}|=|\vec{c}|=1\) かつ \(2\vec{a}+3\vec{b}+\sqrt{7}\,\vec{c}=\vec{0}\) のとき、\(\vec{a}\) と \(\vec{b}\) のなす角を求めよ。
方針3つのベクトルの和が \(\vec{0}\) なので、1つを移項して2乗。
解答
  • \(2\vec{a}+3\vec{b}=-\sqrt{7}\,\vec{c}\) の両辺を2乗：
  • \(|2\vec{a}+3\vec{b}|^2 = 7|\vec{c}|^2\)
  • \(4|\vec{a}|^2 + 12\vec{a}\cdot\vec{b} + 9|\vec{b}|^2 = 7\)
  • \(4 + 12\vec{a}\cdot\vec{b} + 9 = 7\)
  • \(\displaystyle \vec{a}\cdot\vec{b} = -\frac{1}{2}\)
  • \(|\vec{a}|=|\vec{b}|=1\) より \(\displaystyle \cos\theta = -\frac{1}{2}\)
答え\(\displaystyle \theta = \frac{2}{3}\pi\)

③ 終点範囲・軌跡を求める応用

末尾キーワード: 「Pの動く範囲を図示せよ」／「Pの存在領域を求めよ」／「領域を求めよ」

問題文の末尾が「動く範囲・存在領域」のとき。係数の和の条件で図形を判定する。

3A: 基本パターン（暗記）— 平面基本

  • \(\vec{OP} = s\vec{a} + t\vec{b}\) のとき
  • ・\(s, t\) 実数全体 → 全平面
  • ・\(s+t = 1\) → 直線AB
  • ・\(s+t = 1,\ s\geqq 0,\ t\geqq 0\) → 線分AB
  • ・\(s+t \leqq 1,\ s\geqq 0,\ t\geqq 0\) → \(\triangle OAB\) の周および内部
📝 例題
問題\(\vec{OP} = s\vec{a} + t\vec{b}\) について、実数 \(s, t\) が \(s+t \leqq 1,\ s\geqq 0,\ t\geqq 0\) を満たして動くとき、点 P の存在範囲を答えよ。
方針暗記している5パターンに照らし合わせる。
解答
  • \(s+t\leqq 1,\ s\geqq 0,\ t\geqq 0\) は基本パターン④に該当。
答え\(\triangle OAB\) の周および内部

3A2: 基本パターン（暗記）— 空間基本

  • \(\vec{OP} = l\vec{a} + m\vec{b} + n\vec{c}\) のとき
  • ・\(l+m+n = 1\) → 平面ABC
  • ・\(l+m+n = 1\), 各\(\geqq 0\) → \(\triangle ABC\) の周および内部
  • ・\(l+m+n \leqq 1\), 各\(\geqq 0\) → 四面体OABCの周および内部
📝 例題
問題\(\vec{OP} = l\vec{a} + m\vec{b} + n\vec{c}\) について、実数 \(l, m, n\) が \(l+m+n=1,\ l\geqq 0,\ m\geqq 0,\ n\geqq 0\) を満たして動くとき、点 P の存在範囲を答えよ。
方針空間版の暗記パターンに照合。
解答
  • 係数和 \(l+m+n=1\) かつ各 \(\geqq 0\) は、空間の基本パターン③に該当。
答え\(\triangle ABC\) の周および内部

3B: 一般の終点範囲（3手順型）標準

  • \(s\)（または \(t\)）を固定
  • 動く方の変数で線を作る
  • 固定した変数を動かして面に拡大
\(0\leqq a\leqq 1,\ \dfrac{1}{2}\leqq b\leqq 1\) のような長方形領域で使う
📝 例題
問題\(\vec{OP} = a\vec{OA} + b\vec{OB}\) について、\(0\leqq a\leqq 1,\ \dfrac{1}{2}\leqq b\leqq 1\) のとき、点 P の動く範囲を図示せよ。
方針3手順型：(1) b を固定して a を動かす → 線分。(2) b を動かす → 線分が平行移動して帯（平行四辺形）。
解答
  • ① \(b=k\)（定数）と固定すると \(\vec{OP} = a\vec{OA} + k\vec{OB}\)。
  • ② \(0\leqq a\leqq 1\) より、P は始点 \(k\vec{OB}\)、終点 \(\vec{OA}+k\vec{OB}\) の線分上を動く。
  • ③ \(\dfrac{1}{2}\leqq k\leqq 1\) と動かすと、線分が平行移動。
  • よって P の範囲は、\(\vec{OB}/2\) と \(\vec{OB}\) を結ぶ線分を「\(\vec{OA}\) 方向に幅 \(|\vec{OA}|\)」だけ平行移動した平行四辺形。
答え4頂点 \(\dfrac{1}{2}\vec{OB},\ \vec{OB},\ \vec{OA}+\vec{OB},\ \vec{OA}+\dfrac{1}{2}\vec{OB}\) を結ぶ平行四辺形（境界を含む）

3C: \((s+t)\) を作って 3-A に帰着応用

  • \(\displaystyle \vec{OP} = (s+t)\cdot\frac{s\vec{a}+t\vec{b}}{s+t}\) と変形
  • 内側の点 Q は線分AB上を動く
  • P は OQ を \((s+t)\) 倍した点 → 図示
「パラメータについて整理する」の発想
📝 例題
問題\(\vec{OP} = s\vec{OA} + t\vec{OB}\) について、\(s\geqq 0,\ t\geqq 0,\ \dfrac{1}{2}\leqq s+t\leqq 1\) のとき、P の動く範囲を答えよ。
方針\((s+t)\) を作って 3-A に帰着。\((s+t)\) を一度の値に固定して帰着 → 動かして拡大。
解答
  • \(\displaystyle \vec{OP} = (s+t)\cdot\frac{s\vec{OA}+t\vec{OB}}{s+t}\)
  • 内側 \(\displaystyle \vec{OQ} = \frac{s\vec{OA}+t\vec{OB}}{s+t}\) は \(\dfrac{s}{s+t}+\dfrac{t}{s+t}=1\) かつ各 \(\geqq 0\) より線分 AB 上を動く。
  • P は OQ を \((s+t)\) 倍した点。
  • \(s+t = \dfrac{1}{2}\) のとき：線分 AB を原点中心に \(\dfrac{1}{2}\) 倍した線分（A, B の中点を結ぶ）。
  • \(s+t = 1\) のとき：線分 AB そのもの。
  • 両線分の間（および境界）が P の範囲 → 台形（A, B, AB の中点 M_A, M_B を結ぶ領域）。
答え4点 A, B, \(\dfrac{\vec{OA}}{2}\), \(\dfrac{\vec{OB}}{2}\) を結ぶ台形（境界を含む）

3D: 始点を変えて簡単化応用

  • \(\vec{a},\vec{b},\vec{c}\) の係数和が \(a+b+c=1\) 等のとき
  • 始点を A に統一（→\(\vec{AP},\vec{AB},\vec{AC}\) で書き直す）
  • 制約条件も書き換え
  • 3-A に帰着
📝 例題
問題平面上に \(\triangle ABC\) と点 O がある。実数 \(a, b, c\) が \(a\geqq 0,\ b\geqq 0,\ c\geqq 0,\ a+b+c=1\) を満たして動くとき、\(\vec{OP} = a\vec{OA} + b\vec{OB} + c\vec{OC}\) で定まる P の存在領域を求めよ。
方針始点を A に統一すると見通しが立つ。
解答
  • \(\vec{OP} - \vec{OA} = b(\vec{OB}-\vec{OA}) + c(\vec{OC}-\vec{OA})\)（∵ \(a+b+c=1\)）
  • \(\vec{AP} = b\vec{AB} + c\vec{AC},\ b+c\leqq 1,\ b\geqq 0,\ c\geqq 0\)
  • これは 3-A の基本パターン④（\(b+c\leqq 1\) かつ各 \(\geqq 0\)）。
答え\(\triangle ABC\) の周および内部

3E: 斜交座標による解法応用

  • \(\vec{a},\vec{b}\) 方向を斜交軸とみなす
  • \((s, t)\) 平面で不等式を図示
  • 元の図に戻す
答案では使いにくいが、見通しが立てやすい
📝 例題
問題\(\vec{OP} = s\vec{OA} + t\vec{OB}\) について、\(0\leqq s\leqq 1,\ 0\leqq t\leqq 1\) のとき、P の動く範囲を斜交座標で考えよ。
方針\(\vec{OA}, \vec{OB}\) を斜交軸に取り、\((s,t)\) 平面上で領域を作って元の図に戻す。
解答
  • \(s, t\) を直交座標とみなすと、\(0\leqq s\leqq 1,\ 0\leqq t\leqq 1\) は単位正方形。
  • これを \(\vec{OA}\) 方向、\(\vec{OB}\) 方向に取り直す（=斜交軸）。
  • 単位正方形の4頂点 \((0,0),(1,0),(1,1),(0,1)\) は順に \(\vec{0},\ \vec{OA},\ \vec{OA}+\vec{OB},\ \vec{OB}\)。
  • よって P の範囲は4頂点 O, A, A+B, B を結ぶ平行四辺形。
答え4頂点 \(O,\ A,\ \vec{OA}+\vec{OB},\ B\) の平行四辺形（境界含む）

3F: 平面/空間の判別に注意発展

  • \(|\vec{x} - \vec{a}| = r\) → 平面なら円、空間なら球面
  • \((\vec{x} - \vec{a})\cdot(\vec{x} - \vec{b}) = 0\) → 平面なら円、空間なら球面
  • \((\vec{x} - \vec{a})\cdot\vec{n} = 0\) → 平面なら直線、空間なら平面
「答えを円周にしないで球面に」
📝 例題
問題空間内の点 P が \((\vec{p}-\vec{a})\cdot(\vec{p}-\vec{b}) = 0\) を満たすとき、P の軌跡を答えよ。
方針内積の積=0 → AB を直径とする図形。空間か平面かで「球面」「円」が変わるので注意。
解答
  • \((\vec{p}-\vec{a})\cdot(\vec{p}-\vec{b})=0\) は「\(\vec{AP}\cdot\vec{BP}=0\)」
  • つまり ∠APB = 90°（タレスの定理）
  • 空間内なので、AB を直径とする「球面」（円ではない）。
答えAB を直径とする球面

④ 垂線・正射影・外心標準

末尾キーワード: 「垂線の足を求めよ」／「正射影ベクトルを求めよ」／「外心を求めよ」

「垂線・正射影・外心」を見たら正射影公式または内積=0の条件で連立する。

4A: 直線（ベクトル）への正射影標準

  • 公式: \(\displaystyle \vec{OH} = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|^2}\vec{a}\)
  • 内積 \(\vec{a}\cdot\vec{b}\) を計算
  • \(|\vec{a}|^2\) を計算
  • 公式に代入
「ある点から直線に垂線を下ろす」を見たら即この公式
📝 例題
問題\(|\vec{a}|=2,\ |\vec{b}|=3,\ \vec{a}\cdot\vec{b}=2\) のとき、\(\vec{b}\) の \(\vec{a}\) 上への正射影ベクトル \(\vec{OH}\) を求めよ。
方針公式 \(\displaystyle \vec{OH} = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|^2}\vec{a}\) に代入。
解答
  • \(\displaystyle \vec{OH} = \frac{\vec{a}\cdot\vec{b}}{|\vec{a}|^2}\vec{a} = \frac{2}{4}\vec{a} = \frac{1}{2}\vec{a}\)
答え\(\displaystyle \vec{OH} = \frac{1}{2}\vec{a}\)

4B: 垂線の足を交点として求める標準

  • \(\vec{OH} = \vec{OA} + t\vec{AB}\) と置く（直線AB上）
  • \(\vec{PH}\cdot\vec{AB} = 0\)（垂直条件）
  • \(t\) を決定
公式を使わない別解。連立で解ける
📝 例題
問題\(\triangle OAB\) において \(|\vec{a}|=2,\ |\vec{b}|=1,\ \vec{a}\cdot\vec{b}=1\) とする。点 B から直線 OA に下ろした垂線の足を H とするとき、\(\vec{OH}\) を求めよ（公式を使わずに）。
方針\(\vec{OH} = t\vec{a}\) と置き、\(\vec{BH}\cdot\vec{a}=0\) から t を決める。
解答
  • \(\vec{OH} = t\vec{a}\) と置く。
  • \(\vec{BH} = \vec{OH} - \vec{b} = t\vec{a} - \vec{b}\)
  • \(\vec{BH}\cdot\vec{a}=0\)：\((t\vec{a}-\vec{b})\cdot\vec{a} = t|\vec{a}|^2 - \vec{a}\cdot\vec{b} = 4t - 1 = 0\)
  • \(\displaystyle t = \frac{1}{4}\)
答え\(\displaystyle \vec{OH} = \frac{1}{4}\vec{a}\)

4C: 平面への正射影（空間）応用

  • 公式は「ベクトル→ベクトル」なので、まず法線 \(\vec{n}\) に正射影
  • \(\vec{n}\) への正射影 = \(\displaystyle \frac{\vec{n}\cdot\vec{x}}{|\vec{n}|^2}\vec{n}\)
  • 平面 \(\alpha\) への正射影 = \(\displaystyle \vec{x} - \frac{\vec{n}\cdot\vec{x}}{|\vec{n}|^2}\vec{n}\)
📝 例題
問題空間内の点 P\((1,2,3)\) を平面 \(\alpha\): \(x+y+z=0\) に正射影した点 P' の座標を求めよ。
方針公式は「ベクトル→ベクトル」なので、まず法線方向に射影してから引く。
解答
  • 法線 \(\vec{n} = (1,1,1)\)、\(\vec{OP} = (1,2,3)\)
  • \(\vec{n}\cdot\vec{OP} = 1+2+3 = 6,\ |\vec{n}|^2 = 3\)
  • \(\vec{n}\) への射影 \(\displaystyle = \frac{6}{3}\vec{n} = 2(1,1,1) = (2,2,2)\)
  • \(\vec{OP'} = \vec{OP} - (\vec{n}\) への射影\() = (1,2,3) - (2,2,2) = (-1,0,1)\)
答えP'\((-1, 0, 1)\)

4D: 三角形の外心応用

  • \(\vec{OR} = x\vec{a} + y\vec{b}\) と置く
  • 辺の中点 M, N を取る
  • \(\vec{RM}\cdot\vec{OA}=0,\ \vec{RN}\cdot\vec{OB}=0\) を連立
  • \(x, y\) を決定
外心は「垂直二等分線の交点」
📝 例題
問題\(\triangle OAB\) において \(|\vec{a}|=2,\ |\vec{b}|=2,\ \vec{a}\cdot\vec{b}=1\) のとき、外心 R について \(\vec{OR} = x\vec{a}+y\vec{b}\) と置くとき、\(x, y\) を求めよ。
方針辺 OA, OB の中点を M, N とし、RM⊥OA, RN⊥OB を立式して連立。
解答
  • \(\displaystyle M=\frac{\vec{a}}{2},\ N=\frac{\vec{b}}{2}\)
  • \(\vec{RM}\cdot\vec{a}=0\)：\(\displaystyle \left(\frac{\vec{a}}{2}-x\vec{a}-y\vec{b}\right)\cdot\vec{a}=0\)
  • \(\displaystyle \frac{|\vec{a}|^2}{2} - x|\vec{a}|^2 - y\vec{a}\cdot\vec{b} = 0\)
  • \(2 - 4x - y = 0\) … ①
  • 同様に \(\vec{RN}\cdot\vec{b}=0\) より \(2 - x - 4y = 0\) … ②
  • ①②を解いて \(\displaystyle x = y = \frac{2}{5}\)
答え\(\displaystyle x = y = \frac{2}{5}\)

4E: 正射影 × 外心の合わせ技応用

  • 外心から辺へ垂線を下ろすと中点に落ちる
  • 外心と垂直二等分線の相性は◎
📝 例題
問題\(\triangle OAB\) の外心を R とする。R から辺 OA に下ろした垂線の足 H は、辺 OA のどこに位置するか。
方針外心は OA, OB の垂直二等分線の交点。垂直二等分線は OA の中点を通る。
解答
  • R は OA の垂直二等分線上 → 垂線の足 H は OA の中点。
答えH は辺 OA の中点（つまり \(\displaystyle \vec{OH} = \frac{\vec{a}}{2}\)）

4F: 平行六面体の高さ発展

  • 底面の法線方向 \(\vec{n}\)
  • 高さ \(\displaystyle |\vec{OH}| = \frac{|\vec{b}\cdot\vec{n}|}{|\vec{n}|}\)
  • 体積 \(V\) = 底面積 \(\times\) 高さ
📝 例題
問題3つのベクトル \(\vec{a}=(1,0,0),\ \vec{b}=(0,1,0),\ \vec{c}=(1,1,1)\) で張られる平行六面体の高さ（底面を \(\vec{a},\vec{b}\) の平面とする）を求めよ。
方針底面の法線 n を取り、c の n 方向への正射影の長さが高さ。
解答
  • 底面 \(\vec{a},\vec{b}\) の法線方向 \(\vec{n}=(0,0,1)\)
  • \(\vec{c}\cdot\vec{n} = 1,\ |\vec{n}|=1\)
  • \(\displaystyle h = \frac{|\vec{c}\cdot\vec{n}|}{|\vec{n}|} = 1\)
答え高さ \(h = 1\)

⑤ 直線・平面の方程式（空間中心）標準

末尾キーワード: 「直線の方程式を求めよ」／「平面の方程式を求めよ」／「点と平面の距離」

ベクトル方程式を立てる。直線は「点+方向」、平面は「点+2方向」または「点+法線」。

5A: 直線のベクトル方程式基本

  • 点 \(A(\vec{a})\) を通り、方向ベクトル \(\vec{l}\) に平行: \(\vec{x} = \vec{a} + t\vec{l}\)
  • 点 \(A(a,b,c)\) を通り、方向 \((\alpha,\beta,\gamma)\) に平行:
  • \(\displaystyle \frac{x-a}{\alpha} = \frac{y-b}{\beta} = \frac{z-c}{\gamma}\)
📝 例題
問題点 \(A(1,2,3)\) を通り、方向ベクトル \(\vec{l}=(2,-1,1)\) に平行な直線の方程式を求めよ。
方針公式に代入するだけ。
解答
  • \(\displaystyle \frac{x-1}{2} = \frac{y-2}{-1} = \frac{z-3}{1}\)
  • または、パラメータ表示 \((x,y,z) = (1+2t,\ 2-t,\ 3+t)\)
答え\(\displaystyle \frac{x-1}{2} = \frac{y-2}{-1} = \frac{z-3}{1}\)

5B: 平面のベクトル方程式基本

  • 3点 A,B,C を通る: \(\vec{x} = \alpha\vec{a} + \beta\vec{b} + \gamma\vec{c},\ \alpha+\beta+\gamma=1\)
  • 点 A を通り \(\vec{u}, \vec{v}\) で張られる: \(\vec{x} = \vec{a} + s\vec{u} + t\vec{v}\)
  • 点 A を通り、法線 \(\vec{n}\) に平行: \((\vec{x}-\vec{a})\cdot\vec{n} = 0\)
  • 成分形: \(\alpha(x-a) + \beta(y-b) + \gamma(z-c) = 0\)
📝 例題
問題3点 \(A(1,0,0),\ B(0,1,0),\ C(0,0,1)\) を通る平面の方程式を求めよ。
方針パラメータ表示でも法線形でもよいが、ここでは法線形で出す。
解答
  • \(\vec{AB}=(-1,1,0),\ \vec{AC}=(-1,0,1)\)
  • 法線 \(\vec{n}=(\alpha,\beta,\gamma)\) と置き、\(\vec{n}\cdot\vec{AB}=0,\ \vec{n}\cdot\vec{AC}=0\)：
  • \(-\alpha+\beta = 0,\ -\alpha+\gamma = 0\) より \(\vec{n}=(1,1,1)\) が取れる。
  • 点 A を通るので \(1\cdot(x-1)+1\cdot y+1\cdot z = 0\)
  • 整理して \(x+y+z = 1\)
答え\(x+y+z=1\)

5C: 法線ベクトルを求める応用

  • 答案用: \(\vec{n} = (x,y,z)\) と置き、\(\vec{n}\cdot\vec{u}=0,\ \vec{n}\cdot\vec{v}=0\) を連立
  • 裏計算: 外積 \(\vec{u}\times\vec{v}\) を計算
  • \(\vec{u}\times\vec{v} = (u_2 v_3 - u_3 v_2,\ u_3 v_1 - u_1 v_3,\ u_1 v_2 - u_2 v_1)\)
答案では外積を使わず内積=0で書く
📝 例題
問題2つのベクトル \(\vec{u}=(1,2,3),\ \vec{v}=(2,1,0)\) の両方に垂直なベクトルを1つ求めよ（答案用：内積=0で連立）。
方針n=(x,y,z) と置き、u·n=0, v·n=0 を連立。
解答
  • \(\vec{n}=(x,y,z)\) と置く。
  • \(\vec{u}\cdot\vec{n}=x+2y+3z=0\) … ①
  • \(\vec{v}\cdot\vec{n}=2x+y=0\) より \(y=-2x\) … ②
  • ②を①に代入：\(x-4x+3z=0\) より \(z=x\)
  • \(x=1\) と取れば \(\vec{n}=(1,-2,1)\)
  • 裏計算（外積）でも \(\vec{u}\times\vec{v}=(2\cdot 0-3\cdot 1,\ 3\cdot 2-1\cdot 0,\ 1\cdot 1-2\cdot 2)=(-3,6,-3)\) で同じ向き。
答え\(\vec{n}=(1,-2,1)\)（または定数倍）

5D: 直線と平面の交点標準

  • 直線をパラメータ表示 \(\vec{x} = \vec{a} + t\vec{l}\)
  • 平面の方程式に代入
  • \(t\) を解く → 交点
📝 例題
問題直線 \((x,y,z)=(1+t, 2t, -1+3t)\) と平面 \(x+y+z=4\) の交点を求めよ。
方針直線のパラメータ表示を平面の方程式に代入して t を解く。
解答
  • \((1+t)+2t+(-1+3t) = 4\)
  • \(6t = 4\) より \(\displaystyle t = \frac{2}{3}\)
  • 代入して \(\displaystyle \left(\frac{5}{3},\ \frac{4}{3},\ 1\right)\)
答え交点 \(\displaystyle \left(\frac{5}{3},\ \frac{4}{3},\ 1\right)\)

5E: 2直線の交点／ねじれの判定応用

  • 各直線をパラメータ表示
  • 連立して解の有無を見る
  • 解なし → ねじれの位置
📝 例題
問題2直線 \(\ell_1\): \((x,y,z)=(t,t,0)\)、\(\ell_2\): \((x,y,z)=(1-s,\ 0,\ s)\) について、交わるか・ねじれの位置か判定せよ。
方針両式が等しくなる (s, t) があるかを連立で確認。
解答
  • \(t=1-s,\ t=0,\ 0=s\) より \(t=0,\ s=0\)
  • 検算：\(t=1-s\) は \(0=1-0=1\) で矛盾。
  • 連立解なし → ねじれの位置。
答えねじれの位置

5F: 点と直線の距離標準

  • 点から直線へ垂線の足 H を出す（4-A 正射影）
  • 距離 = \(|\vec{PH}|\)
📝 例題
問題点 \(P(2,1,3)\) と直線 \(\ell\): \((x,y,z)=(t,t,t)\) との距離を求めよ。
方針P から \(\ell\) への垂線の足 H を出し、\(|\vec{PH}|\) を求める。
解答
  • \(\ell\) 上の点を \(H=(t,t,t)\) と置く。
  • \(\vec{PH}=(t-2, t-1, t-3)\)、方向 \(\vec{l}=(1,1,1)\)
  • \(\vec{PH}\cdot\vec{l}=0\)：\((t-2)+(t-1)+(t-3)=3t-6=0\) より \(t=2\)
  • \(H=(2,2,2)\)、\(\vec{PH}=(0,1,-1)\)
  • 距離 \(|\vec{PH}|=\sqrt{0+1+1}=\sqrt{2}\)
答え距離 \(\sqrt{2}\)

5G: 点と平面の距離標準

  • 公式: \(\displaystyle d = \frac{|\alpha(x_0-a) + \beta(y_0-b) + \gamma(z_0-c)|}{\sqrt{\alpha^2+\beta^2+\gamma^2}}\)
  • 別解: 平面上の点 A をとり \(\vec{AP}\) の \(\vec{n}\) 方向への正射影の長さ
📝 例題
問題点 \(P(1,2,3)\) と平面 \(2x-y+2z=1\) との距離を求めよ。
方針公式に代入。
解答
  • \(\displaystyle d = \frac{|2\cdot 1 - 2 + 2\cdot 3 - 1|}{\sqrt{2^2+(-1)^2+2^2}} = \frac{|5|}{3} = \frac{5}{3}\)
答え距離 \(\displaystyle \frac{5}{3}\)

5H: 共通垂線・最短距離発展

  • 各直線上に動点 P, Q を置く
  • \(\vec{PQ}\cdot\vec{l_1}=0,\ \vec{PQ}\cdot\vec{l_2}=0\) を連立
  • 最短距離 \(|\vec{PQ}|\)
📝 例題
問題2直線 \(\ell_1\): \((x,y,z)=(t,0,0)\)、\(\ell_2\): \((x,y,z)=(0,s,1)\) の最短距離を求めよ。
方針各直線上に動点 P, Q を取り、PQ が両方向ベクトルに垂直になるように決める。
解答
  • \(P=(t,0,0),\ Q=(0,s,1)\) と置く。
  • \(\vec{PQ}=(-t,s,1)\)、方向 \(\vec{l_1}=(1,0,0),\ \vec{l_2}=(0,1,0)\)
  • \(\vec{PQ}\cdot\vec{l_1}=-t=0\) → \(t=0\)
  • \(\vec{PQ}\cdot\vec{l_2}=s=0\) → \(s=0\)
  • \(\vec{PQ}=(0,0,1)\)、最短距離 \(|\vec{PQ}|=1\)
答え最短距離 \(1\)

⑥ 共線・共面・平行・垂直の判定/証明標準

末尾キーワード: 「同一直線上にあることを示せ」／「同一平面上にあることを示せ」／「垂直であることを示せ」

「示せ・証明せよ」を見たら、共線条件・共面条件・内積=0で書き表す。

6A: 3点が同一直線上基本

  • 条件: \(\vec{AB} = k\vec{AC}\) を満たす実数 \(k\) が存在
  • ⇔ \(\vec{OC} = t\vec{OA} + (1-t)\vec{OB}\)
  • 2つのベクトルを書く
  • 平行関係（実数倍）を示す
📝 例題
問題3点 \(A(1,2,3),\ B(2,4,5),\ C(4,8,9)\) が同一直線上にあるか判定せよ。
方針\(\vec{AB}\) と \(\vec{AC}\) が平行（=実数倍）かを見る。
解答
  • \(\vec{AB}=(1,2,2),\ \vec{AC}=(3,6,6)\)
  • \(\vec{AC} = 3\vec{AB}\) なので、3点は同一直線上にある。
答え同一直線上にある

6B: 4点が同一平面上標準

  • \(\vec{OD} = s\vec{OA} + t\vec{OB} + u\vec{OC},\ s+t+u=1\)
  • ⇔ \(\vec{AD} = x\vec{AB} + y\vec{AC}\)（A 始点）
  • 3つのベクトルを取る
  • 一方を他2つの一次結合で表す
📝 例題
問題4点 \(O(0,0,0),\ A(1,0,0),\ B(0,1,0),\ D(2,3,0)\) が同一平面上にあるか判定せよ。
方針D を \(\vec{OA},\vec{OB}\) の一次結合で表せるか確認。
解答
  • \(\vec{OD}=s\vec{OA}+t\vec{OB}\) と置く：\((2,3,0)=s(1,0,0)+t(0,1,0)\)
  • \(s=2,\ t=3\) で成立。よって D は \(\vec{OA},\vec{OB}\) の一次結合で書ける。
  • 4点は同一平面（z=0 平面）上にある。
答え同一平面上にある

6C: 平行条件・垂直条件（成分）基本

  • 平面: \((a,b)\parallel(c,d) \Leftrightarrow ad - bc = 0\)
  • \((a,b)\perp(c,d) \Leftrightarrow ac + bd = 0\)
  • 空間: \(\vec{u}\parallel\vec{v} \Leftrightarrow \vec{u} = k\vec{v}\)
  • \(\vec{u}\perp\vec{v} \Leftrightarrow \vec{u}\cdot\vec{v} = 0\)
📝 例題
問題2つの空間ベクトル \(\vec{u}=(2,-1,3),\ \vec{v}=(1,4,-1)\) が垂直か判定せよ。
方針u·v=0 か確認するだけ。
解答
  • \(\vec{u}\cdot\vec{v} = 2\cdot 1 + (-1)\cdot 4 + 3\cdot(-1) = 2 - 4 - 3 = -5\)
  • 内積が0でないので垂直ではない。
答え垂直ではない

6D: 直線が平面に直交を示す標準

  • 平面の基底 \(\vec{u}, \vec{v}\) を取る
  • \(\vec{l}\cdot\vec{u}=0\) かつ \(\vec{l}\cdot\vec{v}=0\) を示す
📝 例題
問題直線 \(\ell\): 方向 \(\vec{l}=(1,1,1)\) が、平面 \(\alpha\): \(x+y+z=0\) に直交することを示せ。
方針平面の基底（\(\alpha\) 上の一次独立な2ベクトル）両方と \(\vec{l}\) が垂直であることを示す。
解答
  • 平面 \(\alpha\) 上の一次独立なベクトル：\(\vec{u}=(1,-1,0),\ \vec{v}=(1,0,-1)\)
  • \(\vec{l}\cdot\vec{u}=1-1+0=0\)、\(\vec{l}\cdot\vec{v}=1+0-1=0\)
  • よって \(\vec{l}\) は平面 \(\alpha\) に直交する。
答え\(\vec{l}\) は平面 \(\alpha\) に直交する（証明完了）

6E: 一次独立性の活用応用

  • 同じベクトルを2通りに表したとき
  • \(x\vec{a} + y\vec{b} + z\vec{c} = x'\vec{a} + y'\vec{b} + z'\vec{c}\)
  • ⇒ \(\vec{a},\vec{b},\vec{c}\) 一次独立なら \(x=x',y=y',z=z'\)
係数比較の根拠。連立を解く強力な道具
📝 例題
問題\(\vec{a},\vec{b}\) が一次独立のとき、\(\vec{p} = (s+1)\vec{a} + (t-2)\vec{b} = 3\vec{a} + \vec{b}\) を満たす実数 \(s, t\) を求めよ。
方針一次独立なら係数比較ができる。
解答
  • \(\vec{a},\vec{b}\) は一次独立なので、両辺の係数を比較：
  • \(s+1=3 \Rightarrow s=2\)
  • \(t-2=1 \Rightarrow t=3\)
答え\(s=2,\ t=3\)

6F: 対称性ある条件の式変形証明発展

  • 例: \(\vec{AP}\cdot\vec{BP} + \vec{BP}\cdot\vec{CP} + \vec{CP}\cdot\vec{AP} = 0\)
  • 始点を統一（位置ベクトル化）
  • 平方完成、または \(|\cdots|^2 = \cdots\) の形に
  • 図形的解釈（球面・平面・円）
📝 例題
問題空間内の点 P が \(\vec{AP}\cdot\vec{BP} = 0\) を満たすとき、P の軌跡を求めよ（A, B は定点）。
方針1次形と見せかけて、内積を展開すると 2次式 → 平方完成で球面になる。
解答
  • \(\vec{AP}\cdot\vec{BP}=(\vec{p}-\vec{a})\cdot(\vec{p}-\vec{b})=0\)
  • \(|\vec{p}|^2 - (\vec{a}+\vec{b})\cdot\vec{p} + \vec{a}\cdot\vec{b} = 0\)
  • \(\displaystyle \left|\vec{p} - \frac{\vec{a}+\vec{b}}{2}\right|^2 = \left|\frac{\vec{a}-\vec{b}}{2}\right|^2 = \frac{|\vec{AB}|^2}{4}\)
  • 中心 = AB の中点、半径 = AB/2 → AB を直径とする球面。
答えAB を直径とする球面

⑦ ベクトル方程式の表す図形応用

末尾キーワード: 「式が表す図形を答えよ」／「存在範囲を求めよ」

動点 \(\vec{p}\) の式を見て図形を判定する。まず始点統一、次に式の形（パラメータ／内積／2次形）で分類する。

7A: パラメータ表示基本

  • 形: \(\vec{p} = s\vec{a} + t\vec{b}\)
  • ・パラメータ1つ → 直線
  • ・パラメータ2つ → 平面（空間内）
📝 例題
問題空間内で \(\vec{p} = s\vec{a} + t\vec{b}\)（s, t は実数）が表す図形は何か。\(\vec{a},\vec{b}\) は一次独立とする。
方針パラメータが2つで一次結合 → 平面（原点と a, b で張られる）。
解答
  • パラメータ s, t が独立に実数全体を動く → \(\vec{a},\vec{b}\) で張られる平面。
答え原点 O を通り、\(\vec{a},\vec{b}\) で張られる平面

7B: 内積=一次の形標準

  • 形: \(\vec{a}\cdot(\vec{p} - \vec{q}) = 0\)
  • ・平面: 点 Q を通り、\(\vec{a}\) に垂直な直線
  • ・空間: 点 Q を通り、\(\vec{a}\) に垂直な平面
内積形の1次式 → 直線/平面
📝 例題
問題空間内で \(\vec{a}\cdot(\vec{p}-\vec{q}) = 0\) が表す図形は何か（\(\vec{a}\neq\vec{0}\)）。
方針内積=0 の1次式 → 点 Q を通り \(\vec{a}\) に垂直な「直線」か「平面」か、空間/平面で答えが変わる。
解答
  • \(\vec{p}-\vec{q}\) が \(\vec{a}\) に垂直 → P は Q を通り \(\vec{a}\) に垂直な点の集まり。
  • 空間内なので平面。
答え点 Q を通り \(\vec{a}\) に垂直な平面

7C: 絶対値=定数標準

  • 形: \(|\vec{x} - \vec{a}| = r\)
  • ・平面: 中心 A、半径 \(r\) の円
  • ・空間: 中心 A、半径 \(r\) の球面
📝 例題
問題空間内で \(|\vec{p} - \vec{a}| = 3\) が表す図形は何か。
方針絶対値=定数の形 → 円か球面。空間なので球面。
解答
  • P と A の距離が3 → 中心 A、半径3の球面。
答え中心 A、半径 3 の球面

7D: 内積形の積=0標準

  • 形: \((\vec{x} - \vec{a})\cdot(\vec{x} - \vec{b}) = 0\)
  • ・平面: AB を直径とする円
  • ・空間: AB を直径とする球面
📝 例題
問題平面内で \((\vec{p} - \vec{a})\cdot(\vec{p} - \vec{b}) = 0\) が表す図形は何か。
方針AP·BP=0 → ∠APB=90° → AB を直径とする円（タレス）。
解答
  • \(\vec{AP}\cdot\vec{BP}=0\) より \(\vec{AP}\perp\vec{BP}\) または P=A, P=B。
  • つまり ∠APB = 90°（タレスの定理）。
  • 平面内なので、AB を直径とする円。
答えAB を直径とする円

7E: 一般の2次形（平方完成）応用

  • 形: \(|\vec{p}|^2 + \vec{a}\cdot\vec{p} + b = 0\)
  • 平方完成して \(|\vec{p} - \vec{c}|^2 = R^2\) の形に
  • 円（平面）/ 球面（空間）と判定
2次の項を含むなら円・球面が予想される
📝 例題
問題空間内で \(|\vec{p}|^2 - 2\vec{a}\cdot\vec{p} = 0\) が表す図形を求めよ（\(\vec{a}\neq\vec{0}\)）。
方針2次の項あり → 平方完成。
解答
  • \(|\vec{p}|^2 - 2\vec{a}\cdot\vec{p} = 0\)
  • \(|\vec{p}-\vec{a}|^2 - |\vec{a}|^2 = 0\)
  • \(|\vec{p}-\vec{a}|^2 = |\vec{a}|^2\)
  • 中心 A、半径 |a| の球面。
答え中心 A、半径 \(|\vec{a}|\) の球面

7F: 因数分解で2平面の和に応用

  • 2次式が因数分解できる場合
  • 因数分解 → 2つの1次式の積
  • 「内積=0 または 内積=0」の和事象
📝 例題
問題\((\vec{p}-\vec{a})\cdot(\vec{p}-\vec{b}) \cdot (\vec{p}-\vec{c})\cdot(\vec{p}-\vec{d})=0\) のような形ではなく、\(((\vec{p}-\vec{a})\cdot\vec{n_1})((\vec{p}-\vec{b})\cdot\vec{n_2})=0\) が表す図形は何か。
方針積=0 → どちらかの因数が0 → 2平面（または2直線）の和事象。
解答
  • \((\vec{p}-\vec{a})\cdot\vec{n_1}=0\) または \((\vec{p}-\vec{b})\cdot\vec{n_2}=0\)
  • 前者は A を通り \(\vec{n_1}\) に垂直な平面、後者は B を通り \(\vec{n_2}\) に垂直な平面。
答え2平面 \((\vec{p}-\vec{a})\cdot\vec{n_1}=0\) と \((\vec{p}-\vec{b})\cdot\vec{n_2}=0\) の和（合併）

7G: 空間内の円発展

  • 定石1: 「球面 かつ 平面」と捉える
  • \(|\vec{x}-\vec{a}|=r\) かつ \((\vec{x}-\vec{b})\cdot\vec{n} = 0\)
  • 定石2: パラメータ表示
  • \(\vec{x} = \vec{c} + (r\cos\theta)\vec{p} + (r\sin\theta)\vec{q}\)
📝 例題
問題空間内で \(|\vec{p}|=2\) かつ \(\vec{p}\cdot(0,0,1)=1\) を満たす点 P の軌跡を求めよ。
方針球面 かつ 平面 → 空間内の円。
解答
  • \(|\vec{p}|=2\)：原点中心、半径2の球面
  • \(\vec{p}\cdot(0,0,1)=1\)：z = 1 の平面
  • 球面と平面の交わりで、z=1 平面上で原点真上1の点を中心、半径 \(\sqrt{4-1}=\sqrt{3}\) の円。
答えz=1 平面上、中心 \((0,0,1)\)、半径 \(\sqrt{3}\) の円

7H: 平面/空間の判定に注意発展

  • 同じ式でも「平面」か「空間」かで答えが変わる
  • ・平面 → 円・直線
  • ・空間 → 球面・平面
  • 「答えを円周にしないで球面に」
📝 例題
問題\(|\vec{p}-\vec{a}|=r\) が表す図形を、(1) 平面の問題として、(2) 空間の問題として、それぞれ答えよ。
方針同じ式でも、空間か平面かで「球面」「円」が変わる。
解答
  • (1) 平面の問題：中心 A、半径 r の円
  • (2) 空間の問題：中心 A、半径 r の球面
  • ※ 答えを「円」と書いてしまわないように、空間の問題なら必ず「球面」と書く。
答え(1) 円、(2) 球面

道具集（フロー横断）

各フローで参照される共通の道具をまとめておく。

始点統一の判断基本

  • 条件が対称（A,B,C等価） → 位置ベクトル（始点はどこでも）
  • 1点だけ条件が違う → その点を始点に統一
  • 外心が絡む → 外心を始点（\(|\vec{a}|=|\vec{b}|=|\vec{c}|=R\) が使える）
📝 例題
問題\(\triangle ABC\) の外心を O とする。\(|\vec{OA}|=|\vec{OB}|=|\vec{OC}|=R\) であることを利用して、\(\vec{OA}\cdot\vec{OB}\) を辺の長さ \(c=AB\) と \(R\) で表せ。
方針外心が始点なので |OA|=|OB|=|OC|=R が使える。|OA-OB|² = AB² から立式。
解答
  • \(|\vec{OA}-\vec{OB}|^2 = |\vec{AB}|^2 = c^2\)
  • \(|\vec{OA}|^2 - 2\vec{OA}\cdot\vec{OB} + |\vec{OB}|^2 = c^2\)
  • \(R^2 - 2\vec{OA}\cdot\vec{OB} + R^2 = c^2\)
  • \(\displaystyle \vec{OA}\cdot\vec{OB} = R^2 - \frac{c^2}{2}\)
答え\(\displaystyle \vec{OA}\cdot\vec{OB} = R^2 - \frac{c^2}{2}\)

一次独立／一次結合基本

  • 平面: \(\vec{a},\vec{b}\) 一次独立 \(\Leftrightarrow s\vec{a}+t\vec{b}=\vec{0} \Rightarrow s=t=0\)
  • 任意のベクトル \(\vec{x} = p\vec{a} + q\vec{b}\) で一意に表せる
  • 係数比較ができるのが最大の武器

絶対値の2乗 → 内積基本

  • \(|\vec{a}|^2 = \vec{a}\cdot\vec{a}\)
  • \(|\vec{a} \pm \vec{b}|^2 = |\vec{a}|^2 \pm 2\vec{a}\cdot\vec{b} + |\vec{b}|^2\)
  • 「絶対値を見たら2乗」が定石
📝 例題
問題\(|\vec{a}+\vec{b}|=4,\ |\vec{a}-\vec{b}|=2\) のとき、\(\vec{a}\cdot\vec{b}\) を求めよ。
方針両辺を2乗して内積に変換、引き算で a·b を出す。
解答
  • \(|\vec{a}+\vec{b}|^2 = |\vec{a}|^2 + 2\vec{a}\cdot\vec{b} + |\vec{b}|^2 = 16\) … ①
  • \(|\vec{a}-\vec{b}|^2 = |\vec{a}|^2 - 2\vec{a}\cdot\vec{b} + |\vec{b}|^2 = 4\) … ②
  • ①－②：\(4\vec{a}\cdot\vec{b} = 12\)
  • \(\vec{a}\cdot\vec{b} = 3\)
答え\(\vec{a}\cdot\vec{b} = 3\)

角の二等分線標準

  • \(\angle AOB\) の二等分線方向: \(\displaystyle \frac{\vec{a}}{|\vec{a}|} + \frac{\vec{b}}{|\vec{b}|}\)
  • 向きに注意（外角と区別）

重心の位置ベクトル基本

  • 三角形 ABC の重心: \(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}}{3}\)
  • 四面体 ABCD の重心: \(\displaystyle \vec{g} = \frac{\vec{a}+\vec{b}+\vec{c}+\vec{d}}{4}\)
  • \(\vec{GA}+\vec{GB}+\vec{GC}=\vec{0}\)（始点を重心に取れば）

外積（裏計算用・答案不可）応用

  • 定義: \(\vec{a}\times\vec{b} = (a_2 b_3 - a_3 b_2,\ a_3 b_1 - a_1 b_3,\ a_1 b_2 - a_2 b_1)\)
  • 性質: \(\vec{a}\times\vec{b}\) は \(\vec{a},\vec{b}\) の両方に垂直
  • \(|\vec{a}\times\vec{b}|\) = \(\vec{a},\vec{b}\) が作る平行四辺形の面積
  • 用途: 法線ベクトルの「答え合わせ」
  • 平行六面体の体積 \(V = |\vec{a}\cdot(\vec{b}\times\vec{c})|\)
  • 四面体の体積 = \(V/6\)
  • ⚠ 答案では「内積=0を満たす」と書く
📝 例題
問題\(\vec{a}=(1,2,3),\ \vec{b}=(2,1,0)\) の両方に垂直なベクトル（の1つ）を、外積で求めよ。
方針外積の公式に代入（裏計算）。
解答
  • \(\vec{a}\times\vec{b} = (a_2 b_3 - a_3 b_2,\ a_3 b_1 - a_1 b_3,\ a_1 b_2 - a_2 b_1)\)
  • \(= (2\cdot 0 - 3\cdot 1,\ 3\cdot 2 - 1\cdot 0,\ 1\cdot 1 - 2\cdot 2)\)
  • \(= (-3,\ 6,\ -3)\)
  • 検算：\((-3)\cdot 1+6\cdot 2+(-3)\cdot 3 = -3+12-9 = 0\) ✓
  • ⚠ 答案では「\(\vec{n}=(-3,6,-3)\) は \(\vec{a}\cdot\vec{n}=0,\ \vec{b}\cdot\vec{n}=0\) を満たす」と書く。
答え\(\vec{n}=(-3, 6, -3)\)（または定数倍）

対称性のあるベクトルの座標設定応用

  • 平面: 長さ1の \(\vec{a},\vec{b}\) → \(\vec{a}=(1,0),\ \vec{b}=(\cos\theta,\sin\theta)\)
  • 空間: 長さ1の \(\vec{a},\vec{b},\vec{c}\)
  • → \(\vec{a}=(1,0,0),\ \vec{b}=(\cos\theta,\sin\theta,0),\)
  • \(\vec{c}=(l, m, n)\ [l^2+m^2+n^2=1]\)
  • 設定する文字を最小化するのがコツ
📝 例題
問題平面上に長さ1の \(\vec{a},\vec{b}\) があり、なす角は \(60°\) である。これらを座標設定して、\(|\vec{a}+2\vec{b}|\) を求めよ。
方針\(\vec{a}=(1,0)\)、\(\vec{b}=(\cos 60°,\sin 60°)\) と置く。
解答
  • \(\displaystyle \vec{a}=(1,0),\ \vec{b}=\left(\frac{1}{2},\frac{\sqrt{3}}{2}\right)\)
  • \(\displaystyle \vec{a}+2\vec{b}=(1,0)+(1,\sqrt{3})=(2,\sqrt{3})\)
  • \(|\vec{a}+2\vec{b}|=\sqrt{4+3}=\sqrt{7}\)
答え\(|\vec{a}+2\vec{b}|=\sqrt{7}\)

平面 vs 空間の解法選択標準

  • ベクトルで解く（特に四面体・平行六面体で有効）
  • 座標設定で解く（直交軸が見えるとき）
  • 断面図など幾何的に解く（対称面・対称軸を含む面）
  • 展開図を考える（最短経路など）

ベクトル和 \(s\vec{a}+t\vec{b}+u\vec{c}=\vec{0}\) の利用応用

  • 一文字消去: \(\displaystyle \vec{a} = -\frac{t\vec{b}+u\vec{c}}{s}\)
  • 内積を見る: \(\vec{a}\cdot(s\vec{a}+t\vec{b}+u\vec{c}) = 0\)
  • 対称性を見る
  • 1つ移項して2乗 \(|s\vec{a}|^2 = |-(t\vec{b}+u\vec{c})|^2\)
📝 例題
問題\(|\vec{a}|=|\vec{b}|=|\vec{c}|=1\) かつ \(\vec{a}+\vec{b}+\vec{c}=\vec{0}\) のとき、\(\vec{a}\cdot\vec{b}\) を求めよ。
方針1つを移項して2乗、または直接2乗→展開で内積を取り出す。
解答
  • \(|\vec{a}+\vec{b}+\vec{c}|^2 = 0\)
  • \(|\vec{a}|^2+|\vec{b}|^2+|\vec{c}|^2 + 2(\vec{a}\cdot\vec{b}+\vec{b}\cdot\vec{c}+\vec{c}\cdot\vec{a})=0\)
  • \(3 + 2(\vec{a}\cdot\vec{b}+\vec{b}\cdot\vec{c}+\vec{c}\cdot\vec{a})=0\)
  • 対称性より \(\vec{a}\cdot\vec{b}=\vec{b}\cdot\vec{c}=\vec{c}\cdot\vec{a}\) なので、
  • \(3 + 6\vec{a}\cdot\vec{b}=0\) より \(\displaystyle \vec{a}\cdot\vec{b}=-\frac{1}{2}\)
答え\(\displaystyle \vec{a}\cdot\vec{b}=-\frac{1}{2}\)
