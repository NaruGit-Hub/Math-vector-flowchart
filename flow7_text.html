⑦ ベクトル方程式の表す図形応用

末尾キーワード: 「式が表す図形を答えよ」／「存在範囲を求めよ」

動点pの式を見て図形を判定する。まず始点統一、次に式の形（パラメータ／内積／2次形）で分類する。

7A: パラメータ表示基本

  • 形: p = sa + tb
  • ・パラメータ1つ → 直線
  • ・パラメータ2つ → 平面（空間内）

7B: 内積=一次の形標準

  • 形: a·(p − q) = 0
  • ・平面: 点Qを通り、aに垂直な直線
  • ・空間: 点Qを通り、aに垂直な平面
内積形の1次式 → 直線/平面

7C: 絶対値=定数標準

  • 形: |x − a| = r
  • ・平面: 中心A、半径rの円
  • ・空間: 中心A、半径rの球面

7D: 内積形の積=0標準

  • 形: (x − a)·(x − b) = 0
  • ・平面: ABを直径とする円
  • ・空間: ABを直径とする球面

7E: 一般の2次形（平方完成）応用

  • 形: |p|² + a·p + b = 0
  • 平方完成して |p − c|² = R² の形に
  • 円（平面）/ 球面（空間）と判定
2次の項を含むなら円・球面が予想される

7F: 因数分解で2平面の和に応用

  • 2次式が因数分解できる場合
  • 因数分解 → 2つの1次式の積
  • 「内積=0 または 内積=0」の和事象

7G: 空間内の円発展

  • 定石1: 「球面 かつ 平面」と捉える
  • |x−a|=r かつ (x−b)·n = 0
  • 定石2: パラメータ表示
  • x = c + (r cosθ)p + (r sinθ)q

7H: 平面/空間の判定に注意発展

  • 同じ式でも「平面」か「空間」かで答えが変わる
  • ・平面 → 円・直線
  • ・空間 → 球面・平面
  • 「答えを円周にしないで球面に」
